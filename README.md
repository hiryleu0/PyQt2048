# 2048
## Description
It is simple 2048 created by me to practise python and PyQt5 library.

## Game
Use W, S, A, D keys to play. Using menu you can start new game, exit or change
required value to win!

## Requirements
* PyQt5

## Running
Run command from main directory
```
./main.py
```
